Dependencies:
  TerrariaInjector: https://github.com/ConfuzzedCat/TerrariaInjector/releases/tag/v1.2.1
  Harmony: https://github.com/pardeike/Harmony/releases/tag/v2.4.2.0

How to install:
  Install the latest version of TerrariaInjector and the .Net Framework 3.5 version of Harmony(net35) and drag both in to Terraria's root folder(eg. where Terraria.exe is)
  Then after building this Mod, install in \common\Terraria\Mods\
